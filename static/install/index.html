Install

We strongly recommend using one of the official installation methods. Third party installation guides tend to be out-of-date and often contain misguided advice and errors. If you have trouble with the installation process, ask for help from the #grapheneos Matrix / IRC channel.

GrapheneOS has two officially supported installation methods:

  1. WebUSB-based installation
  2. Command-line interface (CLI) installation

We recommend the WebUSB-based installation.

Only use the CLI installation...

  • If you are proficient with the CLI.
  • If you have proper fastboot and signify packages.

The CLI installation offers a way to install GrapheneOS without trusting our server infrastructure. However, this requires being on an operating system with proper fastboot and signify packages along with understanding the process enough to avoid blindly trusting the instructions from our site. For most users, the WebUSB-based installation approach is no less secure and avoids needing any software beyond a browser with WebUSB support.
